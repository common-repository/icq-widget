== Description ==

This widget allows you to send messages to the author on his website ICQ number straight from the pages of the site.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `icq_widget` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add widget ICQ on your sidebar.
4. be happy!
